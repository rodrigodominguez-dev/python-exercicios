# Python Exercícios

Repositório com exercícios e projetos básicos em Python para prática de lógica de programação.

## Projetos

- Antecessor e sucessor
- Aumento salarial
- Calculadora de desconto
- Calculadora de tinta
- Conversor de dólar
- Conversor de medidas
- Dobro, triplo e raiz quadrada
- Média de aluno
- Mensagem de boas-vindas
- Somando dois números
- Tabuada

## Objetivo

Aprender Python e desenvolver lógica de programação para evolução na área de tecnologia.

## Tecnologias utilizadas

- Python 3
- PyCharm
- GitHub
- VS Code
